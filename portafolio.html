Proyectos

Proyecto ACE (Administrador de Centros Estéticos)

Proyecto ACE es un sistema web para administrar centros estéticos. Incluye módulos para gestión de turnos, control de stock, horarios de especialistas, generación de recibos y facturas, auditorías de tablas, y múltiples reportes e informes automatizados. Fue desarrollado como proyecto final de carrera, implementado en PHP, MySQL, JavaScript y librerías como FullCalendar y Bootstrap.

Tecnologías: PHP, MySQL, JavaScript, FullCalendar, jQuery, Bootstrap
